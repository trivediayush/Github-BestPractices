# Simple HTML CSS Web Project

## 📘 Overview
A basic HTML/CSS static page to practice DevOps version control using Git.

## 🛠️ How to Run
Open `index.html` in any web browser.

## 🧩 Technologies Used
- HTML5
- CSS3

## 🧪 Git Workflow
- Feature branches → PR to `dev` → PR to `main`
- Tags added for releases
